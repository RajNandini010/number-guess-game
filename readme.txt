# Number Guessing Game

A fun and interactive number guessing game built using Python's Tkinter library. The objective is to guess a random number between 0 and 100 within a limited number of attempts.

## Features

- The program randomly selects a number between 0 and 100.
- Players have **10 chances** to guess the correct number.
- Provides feedback on whether the guess is too high, too low, or correct.
- Displays a congratulatory message when the correct number is guessed.
- Displays a game over message if the player runs out of attempts.

## Technologies Used

- **Python**: The game is written in Python.
- **Tkinter**: The graphical user interface is created using Tkinter.

## How to Play

1. Run the Python script.
2. Enter a guess in the input box and press the "Submit Guess" button.
3. The game will give feedback:
   - If your guess is too high, it will display "Your guess is too high."
   - If your guess is too low, it will display "Your guess is too low."
   - If your guess is correct, a congratulatory message will appear.
4. You have a total of 10 chances to guess the correct number.

## Prerequisites
